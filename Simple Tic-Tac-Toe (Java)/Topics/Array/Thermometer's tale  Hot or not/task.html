Create a program that analyzes temperature readings stored in an array. The program should accept an integer 'n' for the number of readings, then read 'n' float values as temperatures. Calculate and print the average temperature, the highest temperature, and the count of readings above 30.0°C. Format float outputs to one decimal place.


Sample Input:
5
25.5
32.1
28.7
35.9
22.3

Sample Output:
Average: 28.9
Highest: 35.9
Above 30.0°C: 2


Sample Input:
4
18.2
22.7
20.5
19.8

Sample Output:
Average: 20.3
Highest: 22.7
Above 30.0°C: 0



Memory limit: 256 MB
Time limit: 5 seconds
